Wa-config is a Web Agency production tool. Build from researches and developpements done by Miguel Monwoo from 2011 to 2022.

== Description ==

WA-Config Monwoo will help with Web Agency jobs like :
* Posting past or current missions managable by skills
* Ensure light human and automatic reviews.
* Launch custom authenticated End to End user tests under production server with existing user accounts (Codeception).
* Backup and optimize the website (mandatory to ensure safe tests launch under production data).
* Extend this plugin to improve those base features.
* Run same instance of this plugin in parallele.

https://www.youtube.com/watch?v=55jOhB5QcB4

[Build by Miguel Monwoo](http://miguel.monwoo.com)

This plugin make your website ready to do advanced human or automatic tests reviews.

__WARNING__ : always test your Backup strategie BEFORE launching codeception tests.
Indeed, we never know what may happen in case of falling tests or unknown side effects.
So alway have a backup ready.
We provide links before the codeception test launch to let you do the kind of light or heavy backup you like.

== Installation ==

Upload the wa-config plugin to your WordPress, activate it, and then visit our 'Review' sub menu under 'WA Config' admin menu.

When all is green, and you humanly argree with it, all is done ;)
